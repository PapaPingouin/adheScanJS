adheScanJS
==========

Rewrite of adheScan with JS / node JS / mongodb

install :

sudo apt-get install nodejs-legacy libnfc-dev npm curl

npm install nfc

npm install socket.io


start
=====

nodejs nfc.js

nodejs local.js
